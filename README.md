## Over View

``` 
- Dataset include: 
  + 50 audio files / people
  + 2s / file 
- Data has been cleaned: noise filtering, sound reinforcement 
- Standardized data: 
  + Sample rate: 44100Hz 
  + Channels: Stereo 
  + Bit Depth: 32 bits
```

## Object

``` 
The purpose of this DATASET is DATA for Speaker Recognition Problem.
```
